# Run the client
To run the client:

1. install rust
2. Build the client with with

  ```
  Cargo build --release
  ```
3. Copy `Config.toml.example` to `Config.toml`
4. Modify `Config.toml` to match your setup.
5.  - Run with:
      ```
      Cargo run --release
      ```
      OR

    - Run the produced executable directly
